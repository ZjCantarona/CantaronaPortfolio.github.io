This course focuses on managing enterprise servers with a DevOps culture. In this course, students learn the essentials of running a DevOps environment in Linux servers, configuring and managing services, and how to monitor and leverage alerting in the DevOps environment by using de facto standard monitoring practices and Ansible as its configuration management; the student will also learn how to collaborate using Git and use Immutable environment using Containerized technologies, and cloud services. This course is also designed to mirror the current real-world scenarios we have and how to solve them..

Create workflows using Infrastructure as code practices to build an enterprise environment.

Evaluate the real-world enterprise scenarios and how to improve the service.

Analyze tools for operating and managing enterprise servers.

Ubuntu Servers

Ubuntu Server is a Linux distribution specifically designed for server environments. It is part of the larger Ubuntu operating system family, which is based on Debian Linux. Ubuntu Server is known for its stability, security features, and ease of use, making it a popular choice for managing enterprise servers.

CentOS Servers

CentOS (Community ENTerprise Operating System) was a free and open-source Linux distribution that aimed to provide a robust, stable, and community-supported platform for managing enterprise servers. It was built from the source code of Red Hat Enterprise Linux (RHEL) and maintained compatibility with RHEL.
